# 🌐 Landing Page Project

A simple and responsive landing page built using **HTML** and **CSS** as part of my frontend learning journey.

---

## 📌 Project Overview

This project is a beginner-friendly implementation of a modern landing page layout.
It focuses on mastering:

* Layout structuring
* Flexbox
* Spacing and alignment
* Typography and colors

---

## 🚀 Features

* 🔷 Header with navigation links
* 🚀 Hero section with call-to-action
* 📦 Information section with cards
* 💬 Quote/testimonial section
* 📢 Call-to-action (CTA) section
* ⚫ Footer

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── style.css
│── README.md
```

---

## 🎯 What I Learned

* How to structure a webpage using semantic HTML
* How to align elements using **Flexbox**
* How to maintain consistent layout using containers
* Importance of spacing, padding, and margins
* Debugging layout issues using borders
